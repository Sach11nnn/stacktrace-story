# StackTrace Story 🎬
> Turn your boring errors into epic cinematic tales

## What is this?
Paste any programming error or stack trace and AI converts 
it into a funny, dramatic, cinematic story explaining 
what went wrong — in English or Hindi!

## Features
- 🎭 Cinematic story generation from any error
- 🇮🇳 Hindi + English language support
- 🖼️ AI generated thematic images (Pollinations.ai)
- 💾 Story history saved (Firebase)
- 📋 Copy & Share stories

## Tech Stack
- React
- Groq API (llama-3.3-70b-versatile)
- Firebase Firestore
- Pollinations.ai (image generation)

## Setup
1. Clone the repo
2. Create .env file (see .env.example)
3. Add your API keys in .env
4. npm install
5. npm start

## Environment Variables
See .env.example for required variables
